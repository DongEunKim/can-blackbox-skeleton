# 단계 4: 폴더 감시 검증

## 실행

```bash
# CAN 로거가 logs/ 에 BLF 생성 중이어야 함
# 터미널 1: can_logger
# 터미널 2: file_watcher
python3 -m src.file_watcher
```

## 동작

- `[logging]` output_dir 폴더를 poll_interval 초마다 스캔
- `.blf` 파일만 대상
- **0바이트 파일 무시**
- **크기 안정 시** (연속 2회 동일) 콜백 호출
- on_new_file 콜백 미지정 시 `[감지] 파일명` 출력
