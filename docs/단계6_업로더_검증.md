# 단계 6: 업로더 검증

## 실행

```bash
# 터미널 1: CAN 로거 (logs/ 에 BLF 생성)
python3 -m src.can_logger

# 터미널 2: 업로더 (감시 + 전송 + 삭제)
python3 -m src.uploader
```

## 동작

- file_watcher로 `[logging]` output_dir 감시
- 신규 안정된 BLF 파일 감지
- StreamManager `append_message`로 전송
- 전송 성공 시 로컬 파일 삭제
- use_mock=true 시 mock_uploads/에 기록
